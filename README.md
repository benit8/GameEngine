# GameEngine
WIP C++ game engine

## Features
- Game states management
- System wide font loader
- Animated sprites
- Custom GUI
	- Basic input widgets
	- Background (static or animated gif)

## Dependencies
- SFML (>= 2.5.0)
- OpenGL
